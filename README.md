# Snap Deck Router API

## Description
Snap Deck Router API is a RESTful web service that provides and routes Marvel Snap decks
from multiple sources using a unified API, inspired by the OpenRouter concept.

## Architecture
Client → Routes → Controller → Service → Repository → Data Source (JSON)

## Tech Stack
- Node.js
- Express.js
- Swagger (OpenAPI)

## Installation
```bash
npm install
npm run dev
